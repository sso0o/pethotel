반려견 동반 숙소 검색 및 예약페이지

#### 개발환경
* 운영체제:  macOS
* IDE: IntelliJ IDEA
* 프로젝트 빌드 도구: Gradle
* 프레임워크: Spring Boot, Spring Security
* 데이터베이스: MySQL
* 템플릿 엔진: Thymeleaf
* ORM: JPA, MyBatis
* 프론트엔드: JavaScript, HTML/CSS
* 버전 관리: Git
***

### 구현 기능
* 회원
  * 가입 / 정보 수정 / 탈퇴 
  
* 숙소 
  * 검색 / 예약 / 결제 / 취소
  
* 숙소 관리자
  * 호텔 등록 / 수정 / 삭제
  * 객실 등록 / 수정 / 삭제
  * 예약 관리 / 객실 배정
  
* 전체 관리자
  * 회원 관리 / 
  * 공통코드 관리
